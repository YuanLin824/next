# 安装

## 使用 npm 安装 (以 windows 为例)

使用 npm 安装 ClaudeCodeCli 不需梯子

### 环境准备

Node 环境安装步骤:

1. [Node 下载地址](https://nodejs.cn/en/download)
2. 根据系统选择对应的 Node 版本, 点击下载按钮
3. 下载完成后, 双击应用程序安装 (安装选项默认即可)

<Image src={Node} alt="Node 下载" width="100%" priority />

Git 环境安装步骤:

1. [Git 下载地址](https://git-scm.com/install/windows)
2. 根据芯片架构点击 `Git for Windows/x64 Setup` 或 `Git for Windows/ARM64 Setup` 进行下载
3. 下载完成后, 双击应用程序安装 (安装选项默认即可)

<Image src={Git} alt="Git 下载" width="100%" priority />

### 环境验证

执行命令后出现版本号即表示环境准备完成

```bash
node --version
# v24.16.0

npm --version
# 11.17.0

git --version
# git version 2.54.0.windows.1
```

### 安装 ClaudeCodeCli

```bash
# 切换国内 npm 镜像源 (默认镜像源可能下载报错)
npm config set registry https://registry.npmmirror.com

# 安装 ClaudeCodeCli
npm install -g @anthropic-ai/claude-code@stable

# 验证 ClaudeCodeCli; 执行 claude --version 后, 出现版本号即表示安装完成
claude --version
# 2.1.169 (Claude Code)
```

### 卸载 ClaudeCodeCli

```bash
# 卸载 ClaudeCodeCli
npm uninstall -g @anthropic-ai/claude-code@stable
```

# 配置

## 配置文件

配置文件类型

1. 全局系统提示 (System Prompt), 对所有项目生效: ~/.claude/CLAUDE.md
2. 全局设置, 如语言, 行为偏好等: ~/.claude/settings.json
3. 自定义斜杠命令目录: ~/.claude/commands/

项目级别配置 (仅对当前项目生效)

1. 项目级系统提示: .claude/CLAUDE.md
2. 项目级设置: .claude/settings.json

配置文件优先级

1. 当前项目: .claude/CLAUDE.md 和 .claude/settings.json
2. 用户主目录: ~/.claude/CLAUDE.md 和 ~/.claude/settings.json
3. 内置默认行为

## 常用命令

### CLI 启动命令

| 命令                       | 说明                         |
| -------------------------- | ---------------------------- |
| `claude`                   | 启动交互式会话               |
| `claude -r`                | 恢复当前目录最近一次会话     |
| `claude -c`                | 继续最近一次会话（全局）     |
| `claude -r <name>`         | 恢复指定名称的会话           |
| `claude -n "name"`         | 命名当前会话                 |
| `claude -p "prompt"`       | 打印模式，直接输出结果后退出 |
| `claude "prompt"`          | 以初始提示词启动会话         |
| `claude -w feature-branch` | 在隔离的 git worktree 中启动 |
| `claude --bare`            | 纯净模式，跳过 hooks/skills  |
| `claude --verbose`         | 详细日志模式                 |
| `claude --debug`           | 调试输出                     |
| `claude --version`         | 显示版本号                   |
| `claude --help`            | 显示帮助信息                 |
| `claude config`            | 打开设置编辑器               |
| `claude mcp`               | 管理 MCP 服务器连接          |

### 常用参数

| 参数                                      | 说明                                      |
| ----------------------------------------- | ----------------------------------------- |
| `--model opus\|sonnet\|haiku`             | 指定模型                                  |
| `--effort high\|medium\|low`              | 设置推理力度                              |
| `--max-turns <N>`                         | 限制最大交互轮数（CI 场景）               |
| `--max-budget-usd <N>`                    | 限制最大花费美元（CI 场景）               |
| `--permission-mode plan`                  | 权限模式：`plan` / `acceptEdits` / `auto` |
| `--allowedTools "Bash(git log *)" "Read"` | 允许特定工具                              |
| `--output-format json`                    | JSON 格式输出（CI 场景）                  |
| `--dangerously-skip-permissions`          | 跳过所有权限提示（谨慎使用）              |

# Skills

Skills 是 Claude Code 的可扩展能力模块。每个 Skill 封装了特定领域的专业知识、工作流程和工具集成，通过在会话中使用 `/<skill-name>` 调用。

## 使用方式

在 Claude Code 终端中输入 `/<skill-name>` 即可触发对应技能：

| 用法                   | 说明                         |
| ---------------------- | ---------------------------- |
| `/<skill-name>`        | 直接调用技能                 |
| `/<skill-name> <参数>` | 带参数调用技能               |
| `/skills`              | 查看当前已安装的所有技能列表 |

> **提示**：使用 `npx skills add <skill-name>` 可安装新技能。部分技能在首次触发时会自动下载安装。
