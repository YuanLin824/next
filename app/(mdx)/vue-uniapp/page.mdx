# UniApp

## 简介

UniApp 是 DCloud 推出的跨端开发框架，基于 Vue.js 语法，一套代码可编译到 iOS、Android、Web（H5）以及微信/支付宝/百度/字节跳动等多个小程序平台。

- **一套代码，多端运行** — 编译到 10+ 平台
- **Vue 语法** — 支持 Vue 2 (uni-app) 和 Vue 3 (uni-app x)
- **丰富的生态** — uni-ui 组件库、uniCloud、插件市场

## 环境搭建

```bash
# 方式一：CLI 创建（推荐用于工程化项目）
# Vue 3 + Vite + JavaScript 模板
npx degit dcloudio/uni-preset-vue#vite my-project

# Vue 3 + Vite + TypeScript 模板 (推荐, 后续配置以 ts 模板为例)
npx degit dcloudio/uni-preset-vue#vite-ts my-project

cd my-project
npm install
npm run dev:mp-weixin   # 微信小程序
npm run dev:h5           # H5
npm run dev:app          # App

# 方式二：HBuilderX 创建（官方 IDE）
# 文件 → 新建 → 项目 → uni-app
```

## 生命周期

### 页面生命周期

```vue
<script setup lang="ts">
import { onLoad, onShow, onReady, onHide, onUnload } from "@dcloudio/uni-app"

// onLoad — 页面加载，只触发一次，可获取路由参数
onLoad((options) => {
  console.log("页面加载", options)
})

// onShow — 页面显示 / 从后台切回前台（每次显示都触发）
onShow(() => {
  console.log("页面显示")
})

// onReady — 页面初次渲染完成
onReady(() => {
  console.log("页面渲染完成")
})

// onHide — 页面隐藏 / 切到后台
onHide(() => {
  console.log("页面隐藏")
})

// onUnload — 页面卸载（redirectTo / navigateBack / 关闭）
onUnload(() => {
  console.log("页面卸载")
})
</script>
```

## 路由跳转与传参

```ts
// uni-app 路由 API（与微信小程序 API 命名一致）
// 注意：这些是 uni.* 全局 API，不是 vue-router

// 保留当前页，跳转
uni.navigateTo({
  url: "/pages/detail/index?id=123&name=hello",
  success: () => console.log("跳转成功"),
})

// 关闭当前页，跳转
uni.redirectTo({ url: "/pages/login/index" })

// 关闭所有页面，打开新页面
uni.reLaunch({ url: "/pages/index/index" })

// 跳转 TabBar 页面
uni.switchTab({ url: "/pages/user/index" })

// 返回上一页
uni.navigateBack({ delta: 1 })

// 组件内跳转 — 使用 navigator 组件
// <navigator url="/pages/detail/index?id=123" open-type="navigate">跳转</navigator>
```

## 条件渲染和列表渲染

## 组件通信

```vue
<!-- ===== 父 → 子：props ===== -->
<!-- Child.vue -->
<script setup lang="ts">
defineProps<{ msg: string; count: number }>()
</script>
<template>
  <view>{{ msg }} — {{ count }}</view>
</template>

<!-- Parent.vue -->
<template>
  <Child msg="Hello" :count="count" />
</template>

<!-- ===== 子 → 父：emit ===== -->
<!-- Child.vue -->
<script setup lang="ts">
const emits = defineEmits<{ (e: "update", value: string): void }>()
```

## 数据请求

```ts
// uni.request 是对各平台网络请求的封装
async function fetchData() {
  const [err, res] = await uni.request({
    url: "https://api.example.com/data",
    method: "GET",
    data: { page: 1, size: 10 },
    header: { "Content-Type": "application/json" },
  })
  if (err) return console.error(err)
  return res.data
}

// 封装请求工具
const BASE_URL = "https://api.example.com"

const request = <T = any>(options: UniApp.RequestOptions): Promise<T> => {
  const token = uni.getStorageSync("token")

  return new Promise((resolve, reject) => {
    uni.request({
      ...options,
      url: `${BASE_URL}${options.url}`,
      header: {
        "Content-Type": "application/json",
        ...(token ? { Authorization: `Bearer ${token}` } : {}),
        ...options.header,
      },
      success: (res) => {
        if (res.statusCode === 200) {
          resolve(res.data as T)
        } else if (res.statusCode === 401) {
          uni.showToast({ title: "登录过期", icon: "none" })
          uni.reLaunch({ url: "/pages/login/index" })
        } else {
          uni.showToast({ title: "请求失败", icon: "none" })
          reject(res)
        }
      },
      fail: (err) => {
        uni.showToast({ title: "网络异常", icon: "none" })
        reject(err)
      },
    })
  })
}

// 文件上传
async function uploadFile(filePath: string) {
  const res = await uni.uploadFile({
    url: `${BASE_URL}/upload`,
    filePath,
    name: "file",
    formData: { type: "avatar" },
  })
  return JSON.parse(res.data)
}
```

## 常见问题

### 1. uni-app 与 Vue 的差异
